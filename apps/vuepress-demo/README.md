# vuepress-demo

`@142vip/vuepress`包的使用Demo演示

## 在线浏览

- Netlify： <https://apps-vuepress-demo.netlify.app>
- Vercel： <https://apps-vuepress-demo.vercel.app>

## 介绍

```bash
# 在项目根目录下载依赖
pnpm i

# 切换到vuepress-demo根目录
cd apps/vuepress-demo

# 本地启动
pnpm dev

# 本地编译
pnpm build
```
